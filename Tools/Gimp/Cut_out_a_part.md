# 🧼 4. Remove background (transparent cutout)

### Steps:

1. **Layer → Transparency → Add Alpha Channel**

2. Select the object (any method above)

3. Press **Ctrl + I** (invert selection)

4. Press **Delete**

👉 Result: Background removed → transparent PNG

***

# 👍 What should you use?

* Just trim image → **Crop Tool**

* Extract rectangle → **Rectangle Select**

* Cut out object → **Free Select / Foreground Select**

* Need transparency → **Add Alpha Channel + Delete**
