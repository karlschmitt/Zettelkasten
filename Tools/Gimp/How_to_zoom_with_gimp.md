### ✅ Method 1 (most common)

* Hold **Ctrl**

* Scroll **mouse wheel**

  * ↑ = zoom in

  * ↓ = zoom out

***

### ✅ Method 2 (if configured differently)

Some setups allow:

* Just **mouse wheel** (no Ctrl)

👉 If that doesn’t work, enable it:

1. Go to **Edit → Preferences**

2. Open **Input Devices → Input Controllers**

3. Check mouse wheel behavior (or try below)

***

# 🛠️ Alternative ways to zoom

### 🔎 Zoom Tool

* Press **Z**

* Click = zoom in

* Ctrl + Click = zoom out

***

### ⌨️ Keyboard

* `+` → zoom in

* `-` → zoom out

* `1` → 100%

* `Shift + Ctrl + J` → fit image to window

***

# ⚙️ Pro tip (super useful)

While zoomed in:

* Hold **Space** → drag to pan around the image

***

# 👍 Recommendation

Use this combo for speed:

👉 **Ctrl + Mouse Wheel + Space drag**

That’s the fastest workflow when editing details.
