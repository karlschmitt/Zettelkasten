## 👍 My recommendation (simple setup)

If you’re building something like a Zettelkasten or learning system:

* Use **Devicon** or **Icons8**

* Download **SVG**

* Keep a folder like:

```
/assets/icons/html.svg
/assets/icons/css.svg
/assets/icons/js.svg
```
