# 20260410224651 Krita Version Information

To find your version information in Krita, the method depends slightly on which operating system you are using:

### 1. On Windows and Linux
1. Open Krita.
2. In the top menu bar, click on **Help**.
3. Select **About Krita**.
4. A window will pop up. The version number (e.g., Krita 5.2.2) will be displayed at the very top of that window.

### 2. On macOS
1. Open Krita.
2. In the Apple menu bar at the top of your screen, click on **Krita**.
3. Select **About Krita**.
4. The version number will be displayed in the popup window.

### 3. The Startup Splash Screen
Every time you launch Krita, a "Splash Screen" appears while the program loads. The version number is printed in the **bottom right corner** (or sometimes the top left, depending on the version design) of that loading image.

### 4. For Detailed System Info (Best for Bug Reports)
If you need more than just the version number (like your graphics driver info or exact build date):
1. Go to **Help**.
2. Select **Show system information for bug reports**.
3. This will open a text window with your Krita version, your operating system details, and your hardware specifications.

### 5. Using the Command Line (Advanced)
If you cannot open the program, you can find the version via your terminal or command prompt by typing:
`krita --version`
