# 20260405170434 PowerShell tree command


***

## 🌳 Equivalent of `tree` in PowerShell

Yes—PowerShell has a direct equivalent:

```powershell
tree
```

👉 It works just like in CMD and shows the folder structure.

***

## 🧩 More “PowerShell-native” Way

If you want something more flexible (and more “PowerShell-like”), use:

```powershell
Get-ChildItem -Recurse
```

or shorter:

```powershell
gci -r
```

***

## 🔍 Example: Show Only Folders in Tree Form

```powershell
Get-ChildItem -Recurse -Directory
```

***

## 🎯 Pretty Tree-Like Output (Advanced)

PowerShell doesn’t natively format output like a visual tree, but you can approximate it:

```powershell
Get-ChildItem -Recurse | Format-Table FullName
```

***

## 💡 Best Option (Real Recommendation)

* ✔️ Use `tree` → quick visual overview

* ✔️ Use `Get-ChildItem` → scripting & automation

***

## ⚡ Pro Tip

Combine with filtering:

```powershell
Get-ChildItem -Recurse -Filter *.java
```

👉 Like: “show me all Java files in this tree”
