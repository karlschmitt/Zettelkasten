# 📅 Week 1 — Foundations

## 🎯 Goal:

Become fluent with Angular Material basics.

***

## Day 1 — Install + Setup

Use Angular CLI:

```bash
ng new material-pro --style=scss
cd material-pro
ng add @angular/material
```

Learn:

* themes

* animations

* typography
